## Стиль брендбука

Цвета (строго):
- Основной: #435364 (тёмно-синий/серый) — для линий и заливки
- Акцент: #ae8c56 (золото) — используется точечно, не более 20% от основного
- Тёмный: #1e252b — для контрастных элементов
- Фон иконок: transparent (будут на белом/светлом фоне #fafafa)

Стиль:
- Чистая геометрия, минимализм, без лишних деталей
- Линии: stroke-width 1.5-2px, stroke-linecap="round", stroke-linejoin="round"
- Скруглённые углы (rx=2-4), мягкие формы
- Отсылка к треугольнику из логотипа (мотив крыши/А-формы) где уместно
- Размер viewBox: 0 0 48 48 (квадратные иконки)
- Без text-элементов внутри SVG

## Формат вывода

Для каждой иконки выведи:
1. Название файла
2. Полный SVG-код
3. Одну строку описания

Все SVG должны быть оптимизированы: без лишних атрибутов, без xmlns если inline,
без комментариев внутри SVG. Каждый файл — самодостаточный.

## Как интегрировать

После генерации SVG, в основном проекте заменим:
- Unicode-символы (✗, ✓, ✎, ⚙) на `<img src="img/icons/...">`
- Текстовые теги работ на теги с иконками
- Номера шагов процесса дополним иконками
