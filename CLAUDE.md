# ARK Construction — Сайт v2 (Сторителлинг, светлая тема)

## О проекте

Сайт ремонтной компании ARK Construction (Москва). Основатель и руководитель — Марина Филатова, дизайнер интерьера.

**v1** (папка `../Marsite/`) — тёмная премиальная тема, оценка 4.1/10.
**v2** (этот проект) — светлая тёплая тема, сторителлинг-формат, баланс «компания + персона».

## Стек

- Чистый HTML5 + CSS3 + ванильный JavaScript (без фреймворков)
- GitHub Pages для хостинга
- BEM-методология для CSS

## Файловая структура

## Дизайн-система

### Цвета (брендбук ARK Construction)
```
Фон основной:      #fafafa   (брендбук)
Фон альтернативный: #f3f2f0  (чередование секций)
Поверхности:        #ffffff   (карточки)
Текст основной:     #1e252b   (брендбук тёмный)
Текст вторичный:    #6b7580
Акцент основной:    #435364   (брендбук primary — кнопки, ссылки, UI)
Акцент hover:       #374757
Акцент light:       rgba(67,83,100,.08)
Золото:             #ae8c56   (брендбук accent — дозированно, ≤20%, только крупный текст)
Бордеры:            #e0e4e8
```

### Типографика
- Заголовки: `Cormorant Garamond` (400, 500, 600, 700)
- Тело: `Inter` (300, 400, 500, 600)

### Ключевые решения
- `border-radius: 12px` для карточек (мягкие скругления)
- Мягкие тени `rgba(67,83,100, .06-.12)` (нейтрально-холодные)
- Чередование фона секций вместо тёмных секций
- Много воздуха, `section-padding: 100px`
- SVG-логотип (A-треугольник с золотым акцентом) в header и footer
- Золото (#ae8c56) только для крупных декоративных элементов (stat values, quote marks)

## Структура секций (Вариант В — сторителлинг)

```
1.  HEADER          — фикс., светлый, backdrop-blur
2.  HERO            — «Ремонт под личным контролем собственника»
3.  PROBLEM-SOLUTION — BAB: 3 боли → «Марина создала ARK» → 3 решения (НОВОЕ)
4.  ABOUT           — Марина + ARK переплетены, фото-placeholder
5.  SERVICES        — 6 видов ремонта + 3 допуслуги + сроки + перечень работ
6.  PROCESS         — 6 шагов timeline
7.  PORTFOLIO       — 3 проекта + каркас before/after
8.  ADVANTAGES      — 6 карточек + сравнительная таблица
9.  REVIEWS         — 3 placeholder-карточки
10. CONTACT         — форма + мессенджеры + инфо
11. FOOTER          — навигация, услуги, копирайт 2026
12. FLOATING BTNS   — WhatsApp + Telegram (fixed, bottom-right)
```
